# Colorize Trade Capture UI

## Goal

- Warm up the simplified UI.
- Keep teal dominant, bronze secondary.
- Use color for hierarchy, categorization, and state only.

## Checklist

- [x] Confirm palette direction and existing color cues
- [x] Replace grayscale tokens with warm neutrals and brand accents
- [x] Colorize key surfaces: upload, review, analysis, chart
- [x] Keep destructive red semantic-only
- [x] Verify typecheck, lint, test, and build

## Palette

- `Primary`: deep teal for primary action, focus, and key hierarchy
- `Secondary`: warm bronze for review surfaces and portfolio context
- `Accent`: pale teal wash for selected controls and optional analysis
- `Destructive`: red reserved for failures and excluded data

## Removed Monotony

- Replaced pure gray neutrals with warm stone and ink tints.
- Reintroduced subtle teal and bronze atmosphere in the page background.
- Gave upload, review, and analysis distinct but restrained surface colors.
- Mapped chart buckets to teal and bronze for faster scanning.
